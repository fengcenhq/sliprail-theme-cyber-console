# sliprail-theme-cyber-console

A high-tech dark theme featuring cyan neon glow and futuristic circuit textures. Its design is inspired by advanced command terminals, providing an immersive digital workspace experience.

## Install Theme

Click the button below to install this theme directly in Sliprail:

[![Install in Sliprail](https://img.shields.io/badge/Install_in_Sliprail-Click_Here-blue?style=for-the-badge)](https://sliprail.fengcen.io/install-extension?id=6b648de7-bdcb-4c05-9bd9-f28ef637d64b)

**Note:** If Sliprail is not installed on your computer, the button will not work. Please download and install Sliprail from the [official website](https://sliprail.fengcen.io) first.

## Features

- **Futuristic Aesthetic**: Cyan neon accents combined with dark circuit-inspired textures.
- **High Contrast**: Optimized for readability and an immersive terminal-like atmosphere.
- **Seamless Integration**: Fully compatible with Sliprail's launcher layout.

## Getting Started

1. Install the theme using the button above or via the Sliprail Store.
2. Open Sliprail Settings and go to the **Themes** section.
3. Select **Cyber Console** from the list of installed themes.
